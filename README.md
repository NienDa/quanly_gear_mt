# Gear Management System

Website quản lý và bán gear máy tính được xây dựng bằng Java Spring Boot.

## Công nghệ sử dụng
- Java Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap

## Chức năng
- CRUD sản phẩm
- Quản lý đơn hàng
- Tìm kiếm sản phẩm
- Phân quyền người dùng

## Cách chạy
1. Clone project
2. Import database ql_gear.sql
3. Config application.properties
4. Run project
